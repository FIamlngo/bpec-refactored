# bpec-refactored
Backpack and Enderchest Mod for Fabric v.1.21.11 (currently only supported version)


**WITH Commands**

/ec for enderchest [synced with the normal enderchest inventory] 

and 

/bp for backpack [27 slots] or /bp 1-6 for backpack [9 - 54 slots] 

the **items in all 54 slots get saved** but are only displayed when typing /bp 6 in the chat
